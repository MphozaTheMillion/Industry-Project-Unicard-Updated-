# **App Name**: CampusID

## Core Features:

- Secure Authentication: Allows students and staff to securely log in using their credentials.
- Digital ID Card Generation: Generates a verifiable digital student or staff ID card upon successful login.
- Profile Data Retrieval: Retrieves student/staff information from registration details to populate the digital ID card.
- Image Capture: Allows users to take a professional picture using the device's camera for their digital ID card.
- Digital ID Card Display: Displays the generated digital ID card with user information and photo.
- Role-Based Access: Conditional rendering logic determines fields displayed and data required based on the userâ€™s role (student or staff).

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey trust and institutionality.
- Background color: Very light blue (#F0F8FF) to provide a clean, uncluttered feel.
- Accent color: Muted purple (#B19CD9) for highlights and call-to-action buttons, providing a modern contrast.
- Font pairing: 'Inter' (sans-serif) for both headings and body text; simple, clean and readable on various devices. Note: currently only Google Fonts are supported.
- Clean and intuitive interface with clear call-to-action buttons for easy navigation.
- Use of simple, professional icons to represent different functionalities.
- Subtle transitions and animations for a smooth user experience when navigating between screens.
